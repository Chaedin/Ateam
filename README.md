### Ateam

그린 컴퓨터 아카데미를 다니면서 팀원 3명과 함께

팀 프로젝트인 향수 쇼핑몰 페이지를 만들었습니다.

VSCODE, ECLIPS프로그램을 사용하였으며

HTML, CSS, Javasctipt를 시작으로

React를 활용하여 퍼블리싱을 마쳤고

MySql을 통한 DB설계를 마치고

Ajax로 데이터를 받아와 페이지를 구성하였습니다.
